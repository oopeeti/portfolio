## Getting Started

### Prerequisites

```sh
npm install npm@latest -g
```

### Installation

   ```sh
   git clone https://github.com/oopeeti/portfolio.git
   ```
   
   ```sh
   cd portfolio
   ```
   
   ```sh
   npm install
   ```
   
### Running Locally
   ```sh
   npm run dev
   ```
